# Адміністраторська панель – документація

Цей файл містить інструкції з розгортання, налаштування та використання адмін‑панелі SvitloSk.
(Вміст скопійовано з `Adminka plans/README.md`.)

---
## Мета
Створити окрему адмін‑панель, доступну лише адміністратору, з функціями:
* Управління користувачами (перегляд, блокування, ролі)
* Перегляд та фільтрація логів парсера
* Налаштування інтеграцій (Supabase, GitHub, API)
* Управління контентом (повідомлення, анонси)
* Аналітика та звіти

---
## Розгортання (локальний варіант)
1. Встановіть Docker та Docker‑Compose.
2. Перейдіть у кореневу теку проєкту `c:\Users\ATom\Desktop\MyCoding\PiTest`.
3. Запустіть:
   ```cmd
   cd "Adminka plans"
   docker compose up -d
   ```
4. Відкрийте браузер за адресою `http://localhost:5173`.

---
## Синхронізація з GitHub
Для локальної роботи можна використати кнопку **Синхронізувати** у адмін‑панелі, яка викликає бекенд‑ендпоінт `/api/admin/sync`. Цей ендпоінт виконує `git pull` у кореневій теці проєкту, оновлюючи код з репозиторію.
